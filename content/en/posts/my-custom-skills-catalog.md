This post only counts skills from my local installed layer: `~/.openclaw/skills`.

So the scope is:

- **Included**: skills I installed or maintain locally
- **Not included for now**: built-in OpenClaw repo skills and skills bundled with the Feishu extension
- **Creation date**: based on local directory creation time, used here as the install / landing date

## Short history

### 2026-03-09: writing, installation, screenshots, publishing

The first wave of skills was focused on a clear goal:

- improve **writing quality**: `humanizer`
- improve **agent initiative and automation**: `proactive-agent`
- solve **skill discovery and inventory**: `awesome-skill-installer`, `ls-skill`
- support **remote collaboration via Feishu**: `feishu-screenshot`
- make **blog publishing** frictionless: `hugo-blog-publish`

This was basically a workbench-building phase: get the fundamentals in place for finding tools, writing better, publishing results, and showing things remotely.

### 2026-03-10: repository workflow layer

The next day was about code hosting workflows:

- `github`
- `gitee`

The pattern is simple: day one established the content and assistant layer, day two filled in the repo operations layer.
