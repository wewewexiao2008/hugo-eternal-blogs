## 介绍

这是一篇测试文章，用来验证自动翻译工作流是否正常工作。当这个文件被推送到主分支时，GitHub Actions 工作流应该会自动将其翻译成中文，并在中文内容目录中创建对应的文件。

## 工作原理

1. 你用任一语言写内容
2. 推送到主分支
3. GitHub Actions 自动检测语言
4. Gemini API 翻译内容
5. 创建翻译文件
6. 自动部署网站

## 功能特点

- 自动语言检测
- 保留所有 Markdown 格式
- 翻译前言（标题、描述）
- 正确处理代码块
- 为每种语言创建对应的文件
